## Key Design Decisions

1. **Port 5173 for Web App**: Standard Vite default, familiar to developers
2. **Port 5001 for Host**: Avoids conflict with common dev servers
3. **NATS on 8080**: WebSocket port, separate from TCP
4. **Lazy Loading**: UserDashboard loaded on-demand, not upfront
5. **Shared React**: Single React instance across host and remote
6. **Mock Fallback**: App works without NATS for development
7. **Turborepo**: Efficient builds with caching
8. **TypeScript**: Type safety across all packages

## Performance Considerations

- **Code Splitting**: MFE loaded only when needed
- **Shared Dependencies**: React/React-DOM shared, not duplicated
- **Build Caching**: Turborepo caches unchanged packages
- **WebSocket**: Persistent connection for real-time updates
- **DuckDB WASM**: Client-side analytics, no server needed
